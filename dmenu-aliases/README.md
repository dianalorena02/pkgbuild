# Readme
Run dmenu with aliases support using the `dmenu_run_aliases` script:

```bash
dmenu_run_aliases -fn 'Noto Sans-10' -sb '#3A81CC' -nb '#ffffff' -nf '#000000'
```
Aliases can bei either stored in `~/.bash_aliases` or `~/.zsh_aliases`.
